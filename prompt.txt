## TECHNICAL REQUIREMENTS

- Pure HTML + CSS. No npm, no build tools, no frameworks.
- Load fonts from Google Fonts (DM Sans + Inter).
- AOS from CDN for scroll animations (data-aos="fade-up" on section headings
  and card groups, initialize with duration:700, once:true).
- Responsive: mobile-first, all multi-col grids collapse below 768px.
- All img tags: width + height attributes matching post-optimized dimensions,
  loading="lazy" except hero logo.
- CSS custom properties for all theme values.
- Validate: zero href values contain "gamma.app".
- Orange italic effect: use <em class="accent"> with CSS color: var(--color-accent-orange)
  font-style: italic applied to the relevant words in H1s.
- The callout box in Section 4 should have a navy left border and bold text.
- Blockquotes: styled with orange left border, italic text, attribution in
  small normal weight text below.
- For the 5-stage educational continuum (Section 12), use a horizontal flex
  layout on desktop with numbered steps or a visual progression indicator.
